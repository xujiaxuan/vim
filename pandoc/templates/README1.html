1 web server using python

  • python直接写个完整的服务器，类似web_server目录下的web.py
  • 使用httpd作为服务器，cgi执行具体业务，类似httpd_cgi目录下的hello.py
  • 使用python 编写的web框架django、flask等， flask的例子在flask目录下

1.1 python写完整的服务器

参考web_server目录下的web.py  
1. 执行方法：python web.py。  
2. 访问方式：http://172.16.0.37:8000

注意如果没有static目录可以尝试手动创建

1.2 httpd作为服务器

参考httpd_cgi目录下的hello.py
1. yum install httpd
2. service httpd start

1.2.1 问题

  1. httpd : htppd : bad user name apache

    useradd apache  
sudo useradd -g apache apache

  2. httpd: Could not reliably determine the server's fully qualified domain name修改/etc/httpd/conf/httpd.conf 添加 ServerName 172.16.0.37:80

  3. 不支持cgi及.py
    修改/etc/httpd/conf/httpd.conf添加AddHandler cgi-script .cgi .py
    cgi-bin位置：/var/www/cgi-bin

  4. 如何访问cgi程序
    示例http://172.16.0.37/cgi-bin/hello.py hello.py权限755

    #!/usr/bin/python
# -*- coding: UTF-8 -*-
print "Content-type:text/html"
print                               # 空行，告诉服务器结束头部
print '<html>'
print '<head>'
print '<meta charset="utf-8">'
print '<title>Hello Word - 我的第一个 CGI 程序！</title>'
print '</head>'
print '<body>'
print '<h2>Hello Word! 我是来自菜鸟教程的第一CGI程序</h2>'
print '</body>'
print '</html>'

    部署示例： 172.16.0.37 可以通过云桌面直接访问
    注意hello.py是新增的内容

1.3 使用python编写的web框架flask

1. 安装flask及其依赖：sudo apt-get install python-flask Jinja2
2. 编写服务器脚本 dpi_web_server.py
3. 启动server: python dpi_web_server.py
4. 访问示例：http://172.16.0.37:5000/

1.3.1 flask 插件安装示例：

1. 下载flask-login https://pypi.python.org/pypi/Flask-Login
2. 解压
3. 执行python setup.py install

login可能还需要使用：python-flask-sqlalchemy.noarch

1.3.2 ajax使用示例：

  1. html中的javascript/jquery添加
$.ajax({
    url:'/get_chart_data',
    type:'post',
    data: {type: 'flowchart_10', bridge: 0},
    success:function(json){
        piechart(json, "pieCanvas2", "最近10分钟流量分布");
    }
});
  1. web服务器dpi_web_server.py添加
@app.route('/get_chart_data',methods=['POST','GET'])
def get_chart_data():
if request.method == 'POST':
    chart = request.form.get('type','flowchart_10')
    if chart == "flowchart_10":
        return '{first 3 red,second 1 blue,third 2 green,four 4 blue,}'
    elif chart == "flowchart_all":
        return '{A 4 red,B 4 blue,C 1 green,D 1 blue,}'
    elif chart == "flowchart_flowcnt":
        return '{AA 3 red,BB 3 blue,CC 3 green,DD 1 blue,}'
    else:
        return '{AA 3 red,BB 3 blue,CC 3 green,DD 1 blue,}'
return '{AA 4 red,BB 3 blue,C 3 green,D 4 blue,}'
  1. 访问：http://172.16.0.37:5000/panabit_main

1.3.3 chartkick+flask示例：

1. 安装chartkick
    下载： https://github.com/mher/chartkick.py
    安装: python setup.py install
2. 编写文件：
        ├── run.py   //server
        ├── static
        │   └── js
        │       ├── chartkick.js  //画图脚本
        │       ├── highcharts.js //依赖脚本
        │       └── jquery.min.js //依赖脚本
        └── templates
            └── index.html //页面模板
3. 开启服务器 python run.py
4. 访问http://172.16.0.37:5000

1.3.4 python 生成png示例

python 生成折线图的png图片(参考panabit)用于web界面显示：
    1. 安装pychart
    2. 添加python脚本 例如文件 flask/example/pychart_png/pychart_example.py
    3. 执行python 脚本生成文件 e.png: python pychart_example.py --format=png >e.png
    4. html添加img 路径执行该文件
